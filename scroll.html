ScrollReveal 動畫

可以輕鬆地將滾動進場的動畫效果添加到元素上，讓網頁更具動感。
ScrollReveal 官網

範例程式碼


        <script src="https://unpkg.com/scrollreveal"></script>
    
        // 初始化 ScrollReveal
        ScrollReveal().reveal('.section', {
            duration: 800,
            distance: '50px',
            easing: 'ease-out',
            origin: 'bottom',
            interval: 100,
            reset: true
        });
